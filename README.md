# VoxCPM-Voice-TTS

Voice agent TTS platform featuring custom voice training via VoxCPM LoRA configurations.

![Python](https://img.shields.io/badge/Python-3.11-blue?logo=python)
![FastAPI](https://img.shields.io/badge/FastAPI-0.110-green?logo=fastapi)
![PyTorch](https://img.shields.io/badge/PyTorch-AI-orange?logo=pytorch)

## Overview
VoxCPM-Voice-TTS handles voice generation and training. It exposes Text-to-Speech routes and includes Jupyter notebooks mapping training setups for custom VoxCPM voice models.

## Features
- **VoxCPM TTS Engine** — Generates natural audio using specific voice parameters.
- **LoRA Training Pipeline** — Colab training setups for custom voices.
- **Database logs** — Cataloging voice models and user queries.

## Tech Stack
| Layer | Technology |
|-------|-----------|
| Backend | FastAPI, Python 3.11, SQLite |
| ML Frameworks | PyTorch, VoxCPM Voice Engine |
